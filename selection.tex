\subsection{Signal Region Selections}

The signal region that contain $N_{jets}>=2$ jets is defined using cuts at the reconstructed level objects, considered for the measurement of ggF production. The difinition of the signal region is based on the selections used for Run 1 analysis \cite{2015}.
